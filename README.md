## Simple Arcade Action Built with Godot

This repository contains the source code for **Gem Catcher**, a fast-paced, arcade-style game developed as a requirement for academic studies in game development.

The core objective is simple: use a movable paddle to catch valuable falling gems while avoiding hazards, earning points, and surviving progressively challenging rounds. It's designed for quick, replayable fun!

---

## üåü Features

- **Arcade-Style Gameplay:** Simple controls and immediate action optimized for quick sessions.  
- **Gem Collection:** Catch different types of falling gems to earn varying point values.  
- **Paddle Control:** Intuitive horizontal control over the player's paddle object.  
- **Progressive Difficulty:** Gems and hazards drop faster and more frequently over time.  
- **Scoring System:** Track your high score across multiple attempts.
